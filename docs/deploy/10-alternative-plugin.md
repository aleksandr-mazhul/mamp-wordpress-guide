# 10. Альтернатива: перенос через плагин

[← Оглавление части 2](README.md)

Если не хотите возиться с FTP и phpMyAdmin — можно перенести сайт **плагином**. Основной гайд ([01–09](README.md)) — **ручной путь** для обучения. Этот раздел — короткая альтернатива.

---

## Когда плагин имеет смысл

| Подходит | Не подходит |
|----------|-------------|
| Маленький учебный сайт | Большой сайт с гигабайтами медиа |
| Нужно быстрее | Хотите понять устройство WP |
| Бесплатный лимит плагина хватает | Нужен перенос без ограничений |

---

## All-in-One WP Migration

Сайт плагина: [wordpress.org/plugins/all-in-one-wp-migration](https://ru.wordpress.org/plugins/all-in-one-wp-migration/)

### На локальном сайте (Mac)

1. MAMP → Start
2. Админка: `http://localhost/название-вашей-папки/wp-admin/`
3. **Плагины** → **Добавить новый** → **All-in-One WP Migration**
4. **Установить** → **Активировать**
5. **All-in-One WP Migration** → **Экспорт** → **В файл**
6. Скачается файл `.wpress`

### На хостинге

1. Установите чистый WordPress на хостинге **или** загрузите только файлы WP + создайте пустую БД (зависит от хостинга)
2. Установите тот же плагин в админке хостинга
3. **Импорт** → загрузите `.wpress`

### Ограничения (важно!)

- **Бесплатная версия** имеет **лимит размера** импорта (проверьте на сайте плагина — лимит меняется)
- Для файлов **больше лимита** нужно **платное расширение** (Import Extension)
- На бесплатном хостинге лимиты загрузки могут дополнительно мешать

> Для учебного сайта с несколькими страницами часто хватает бесплатной версии.

## Duplicator

Сайт: [wordpress.org/plugins/duplicator](https://ru.wordpress.org/plugins/duplicator/)

Похожий принцип: пакет (архив + installer) → загрузка на хостинг → мастер установки.

- Бесплатная версия — для простых сайтов
- Pro-функции — платно

## Сравнение с ручным путём

| | Ручной (01–09) | Плагин |
|---|----------------|--------|
| Обучение | Высокое | Низкое |
| Скорость | Дольше | Быстрее |
| Стоимость | Бесплатно | Может потребоваться платный импорт |
| Универсальность | Любой хостинг | Зависит от лимитов |
