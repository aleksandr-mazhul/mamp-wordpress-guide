# 09. Финальная проверка

Сайт перенесён — осталось убедиться, что всё работает.

---

## Чеклист

Пройдитесь по пунктам:

- [ ] **Главная страница** открывается по адресу хостинга
- [ ] **Вход в админку** (`/wp-admin/`) работает
- [ ] **Страницы и записи** открываются (не только главная)
- [ ] **Картинки** отображаются (не «битые» иконки)
- [ ] **Меню** на месте
- [ ] **Постоянные ссылки** — откройте любую запись, URL красивый (не `?p=123`)

Если пункт с ссылками не работает: **Настройки** → **Постоянные ссылки** → **Сохранить**.

---

## HTTPS (если хостинг выдал SSL)

1. Откройте сайт с `https://`
2. **Настройки** → **Общие**
3. Поля **Адрес WordPress** и **Адрес сайта** — оба с `https://`
4. Сохраните

> На бесплатных хостингах SSL может включаться в панели отдельно — смотрите раздел SSL / Let's Encrypt.
