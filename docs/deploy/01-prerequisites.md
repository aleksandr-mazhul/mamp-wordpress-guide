# 01. Подготовка к переносу

[← Оглавление части 2](README.md) | [Далее: Выбор хостинга →](02-choose-hosting.md)

Перед переносом убедитесь, что всё готово. Это сэкономит время и нервы.

---

## Что должно быть готово

### 1. Рабочий локальный сайт

Вы прошли [Часть 1](../06-first-launch.md). Сайт открывается по адресу:

```
http://localhost/название-вашей-папки/
```

Заполните таблицу (из раздела 06) — она понадобится при переносе:

| Что | Ваше значение |
|-----|----------------|
| Папка с файлами | `/Applications/MAMP/htdocs/...` |
| URL сайта | `http://localhost/...` |
| Имя базы данных | |
| Логин WordPress | |
| Пароль WordPress | |

### 2. MAMP запущен

Для экспорта базы понадобится запущенный MAMP (Apache + MySQL — зелёные).

### 3. Аккаунт на хостинге

Зарегистрируйтесь на выбранном хостинге (см. [следующий раздел](02-choose-hosting.md)). Сохраните:

- логин и пароль от **панели управления** хостингом
- адрес сайта (домен или субдомен), например `mysite.infinityfreeapp.com`

### 4. Доступ к файлам на хостинге

Нужен **один** из способов:

| Способ | Что это |
|--------|---------|
| **Файловый менеджер** | Загрузка файлов через браузер в панели хостинга |
| **FTP** | Программа на Mac для передачи файлов на сервер |

Для FTP установите бесплатный **FileZilla**: [filezilla-project.org](https://filezilla-project.org/download.php?type=client)

В панели хостинга найдите данные FTP:

- хост (например `ftpupload.net` или `ftp.ваш-домен.com`)
- логин FTP
- пароль FTP
- порт (обычно `21`)

## Почему перенос — это нормально

| | Локально (MAMP) | На хостинге |
|---|-----------------|-------------|
| Кто видит сайт | Только вы | Любой в интернете |
| Адрес | `localhost` | Домен хостинга |
| Зачем | Разработка, учёба | Публикация |

Локальный сайт — черновик. Хостинг — публикация в интернет.

---

## Что понадобится на Mac

- Браузер
- Архиватор (встроенный в macOS — достаточно)
- FileZilla (если используете FTP)
- 30–60 минут спокойного времени

---

## Не паникуйте, если…

- Панель хостинга выглядит иначе, чем на скриншотах — ищите **похожие названия** (Базы данных, FTP, Файловый менеджер)
- Хостинг на английском — кнопки часто называются **MySQL Databases**, **File Manager**, **phpMyAdmin**
