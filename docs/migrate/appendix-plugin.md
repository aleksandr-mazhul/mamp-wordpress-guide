## Сделайте (All-in-One WP Migration)

### На Mac

1. MAMP → **Start**
2. Браузер → `http://localhost/название-вашей-папки/wp-admin/`
3. **Плагины** → **Добавить новый** → найдите **All-in-One WP Migration**
4. **Установить** → **Активировать**
5. Слева **All-in-One WP Migration** → **Экспорт** → **В файл**
6. Скачается файл `.wpress` — сохраните на Mac

### На хостинге

7. Зарегистрируйтесь на хостинге ([как в шаге 2](02-hosting.md), если ещё нет аккаунта)
8. Установите чистый WordPress на хостинге ([Часть 3](../hosting/03-wordpress.md) — только установка файлов и БД)
9. Войдите в админку хостинга → установите тот же плагин
10. **All-in-One WP Migration** → **Импорт** → загрузите `.wpress`

**Проверка:** сайт на домене хостинга с тем же контентом, что был на Mac.

## Пояснение

<details>
<summary>Ограничения (важно)</summary>

- **Бесплатная версия** имеет **лимит размера** — см. [сайт плагина](https://ru.wordpress.org/plugins/all-in-one-wp-migration/)
- Большие сайты — платное расширение Import
- Бесплатный хостинг может дополнительно ограничивать размер загрузки
</details>

<details>
<summary>Когда плагин, когда ручной путь</summary>

| | Ручной 1–6 | Плагин |
|---|------------|--------|
| Обучение | Высокое | Низкое |
| Скорость | Дольше | Быстрее |
| Большой сайт | Да | Лимиты / платно |

**Рекомендация:** первый раз — [ручной путь](01-prepare.md).
</details>
