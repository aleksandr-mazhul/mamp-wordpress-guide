Основной путь — **ручной** ([README](README.md)). Плагин — если сайт маленький и не хотите возиться с SQL и FTP.

---

## Сделайте (All-in-One WP Migration)

### На Mac

1. MAMP → Start → локальная админка
2. Плагины → Добавить → **All-in-One WP Migration** → Установить → Активировать
3. Экспорт → В файл → скачается `.wpress`

### На хостинге

4. Установите WordPress на хостинге (или файлы + пустая БД)
5. Тот же плагин → Импорт → загрузите `.wpress`

**Проверка:** сайт открывается на домене хостинга с тем же контентом.

## Пояснение

<details>
<summary>Ограничения (важно)</summary>

- **Бесплатная версия** имеет **лимит размера** импорта — проверьте на [сайте плагина](https://ru.wordpress.org/plugins/all-in-one-wp-migration/)
- Файлы больше лимита — нужно **платное расширение**
- На бесплатном хостинге лимиты загрузки могут дополнительно мешать
</details>

<details>
<summary>Duplicator</summary>

Похожий принцип: архив + installer → мастер на хостинге. Бесплатно для простых сайтов, Pro — платно.
</details>

<details>
<summary>Когда плагин, когда ручной путь</summary>

| | Ручной | Плагин |
|---|--------|--------|
| Обучение | Высокое | Низкое |
| Скорость | Дольше | Быстрее |
| Большой сайт | Да | Лимиты / платно |

**Рекомендация:** первый раз — [ручной гайд](README.md).
</details>
