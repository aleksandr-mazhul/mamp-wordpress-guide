## Сделайте

> Названия кнопок на разных хостингах отличаются — ищите **похожие** слова на русском или английском.

### Регистрация

1. Откройте сайт хостинга, например [InfinityFree](https://www.infinityfree.com/) или [AwardSpace](https://www.awardspace.com/)
2. Нажмите **Sign Up** / **Регистрация** / **Create Account**
3. Заполните email, пароль, подтвердите регистрацию (если пришло письмо — перейдите по ссылке)

### Вход в панель

4. На сайте хостинга нажмите **Login** / **Вход** / **Client Area**
5. Введите логин и пароль → откроется **панель управления** (vPanel, cPanel, hPanel — название не важно)
6. Если хостинг просит **создать сайт** — создайте: выберите субдомен (например `mysite.infinityfreeapp.com`) или привяжите домен

### Запишите данные

7. На главной панели найдите **адрес вашего сайта** и запишите:

| Поле | Ваше значение |
|------|----------------|
| URL сайта | `http://...` или `https://...` |
| Логин панели хостинга | |
| Пароль панели | |

### Найдите нужные разделы (пока не нажимайте «создать»)

8. Откройте раздел **MySQL Databases** / **Базы данных** — запомните, где он
9. Найдите **File Manager** / **Файловый менеджер**
10. Найдите кнопку **phpMyAdmin** (часто в том же блоке, что и MySQL)

## Пояснение

<details>
<summary>Минимальные требования хостинга</summary>

PHP 8.0+, MySQL/MariaDB, phpMyAdmin, File Manager. Для учёбы подойдёт бесплатный тариф.
</details>

<details>
<summary>Где искать, если панель на английском</summary>

| Ищите | Зачем |
|-------|-------|
| MySQL Databases | Создать базу (шаг 4) |
| File Manager | Загрузить файлы (шаг 3) |
| phpMyAdmin | Импорт SQL (шаг 4) |
| FTP Accounts | Только если пойдёте по [appendix-ftp](appendix-ftp.md) |
</details>
