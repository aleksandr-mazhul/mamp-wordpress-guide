## Сделайте

Пройдитесь по чеклисту:

- [ ] Главная открывается по адресу хостинга
- [ ] Вход в `/wp-admin/` работает
- [ ] Страницы и записи открываются (не только главная)
- [ ] Картинки на месте
- [ ] Меню на месте
- [ ] Постоянные ссылки красивые (не `?p=123`)

Если ссылки не работают: **Настройки** → **Постоянные ссылки** → **Сохранить**.

### HTTPS (если есть SSL)

1. Откройте сайт с `https://`
2. Настройки → Общие — оба URL с `https://`
3. Сохраните

**Проверка:** все пункты чеклиста отмечены — сайт в интернете.

## Пояснение

<details>
<summary>Что изменилось</summary>

| | Было (Mac) | Стало (хостинг) |
|---|------------|-----------------|
| URL | `localhost/папка/` | домен хостинга |
| Файлы | `htdocs/` | `public_html` |
| БД | `root`/`root` | пользователь из панели |
| Кто видит | Только вы | Все в интернете |
</details>

<details>
<summary>Локальный сайт</summary>

MAMP-сайт на Mac **не удаляется** — удобно для правок и повторных переносов.
</details>
