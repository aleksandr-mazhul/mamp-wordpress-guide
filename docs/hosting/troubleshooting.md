<a id="db-connection"></a>

## Error establishing a database connection

Проверьте данные из [шага 2](02-database.md) при установке WordPress:

| Поле | Частая ошибка |
|------|----------------|
| DB_NAME | Не полное имя из панели |
| DB_USER / DB_PASSWORD | Опечатка |
| DB_HOST | `localhost` по привычке — **возьмите из панели** |

<a id="page-404"></a>

## Страница не найдена (404)

1. Файлы в `public_html`, есть `index.php`
2. URL совпадает с доменом из панели
3. После установки: **Настройки** → **Постоянные ссылки** → **Сохранить**

---

<a id="permalinks-404"></a>

## 404 на страницах кроме главной

**Настройки** → **Постоянные ссылки** → **Сохранить**. Проверьте `.htaccess` в корне.

---

<a id="white-screen"></a>

## Белый экран

В `wp-config.php`: `WP_DEBUG` и `WP_DEBUG_DISPLAY` = `true`. Часто виноват плагин — отключите через File Manager.

- [ ] `index.php` в корне `public_html`

- [ ] Постоянные ссылки сохранены
