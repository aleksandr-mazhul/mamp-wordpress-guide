# 02. Установка MAMP

[← Подготовка](01-prerequisites.md) | [Назад к оглавлению](../README.md) | [Далее: Настройка MAMP →](03-configure-mamp.md)

Установим MAMP Free на Mac. Нужна именно **бесплатная версия MAMP**, не MAMP PRO.

---

## Шаг 1. Скачать MAMP

1. Откройте **[mamp.info/ru/downloads](https://www.mamp.info/ru/downloads/)**
2. Найдите блок **MAMP** (бесплатная версия)
3. Нажмите **Download** — скачается файл `.pkg`

> Если сайт предлагает MAMP PRO — прокрутите ниже или найдите кнопку для бесплатной версии.

---

## Шаг 2. Установить

1. Откройте скачанный файл `MAMP_*.pkg` (двойной клик)
2. Следуйте мастеру: **Продолжить** → **Продолжить** → **Принимаю** → **Установить**
3. Введите пароль администратора Mac
4. Дождитесь завершения и нажмите **Закрыть**

MAMP установится в `/Applications/MAMP/`.

---

## Шаг 3. Первый запуск

1. Откройте **Finder** → **Программы** (Applications)
2. Запустите **MAMP** (не MAMP PRO)

### Если macOS блокирует запуск

1. **Системные настройки** → **Конфиденциальность и безопасность**
2. Найдите сообщение о MAMP → **Всё равно открыть**

---

## Проверка

Должно открыться главное окно MAMP. Если окно появилось — установка прошла успешно. Переходите к настройке.

> **Не нажимайте Start пока** — сначала настроим Apache, порты и папку для сайта в следующем разделе.
