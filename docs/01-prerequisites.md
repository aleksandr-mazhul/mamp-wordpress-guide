## Что скачать

### 1. MAMP (бесплатная версия)

Скачайте **MAMP Free** с официального сайта:

**[mamp.info/ru/downloads](https://www.mamp.info/ru/downloads/)**

На странице выберите **MAMP** (не MAMP PRO). Бесплатная версия устанавливается по умолчанию.

### 2. WordPress

Скачайте последнюю версию WordPress на русском:

**[ru.wordpress.org/download](https://ru.wordpress.org/download/)**

Скачается архив `wordpress-x.x.x-ru_RU.zip`. Сохраните его — распакуем позже.

> Скачивайте только с официальных сайтов. Сторонние «сборки» могут содержать вредоносный код.

## Что такое MAMP?

**MAMP** (Mac, Apache, MySQL, PHP) — программа, которая превращает ваш Mac в локальный веб-сервер:

- **Apache** — веб-сервер, отдаёт страницы в браузер
- **MySQL** — база данных, хранит контент сайта
- **PHP** — язык, на котором написан WordPress

---

## Локальный WordPress vs хостинг

| | Локально (MAMP) | На хостинге |
|---|-----------------|-------------|
| Доступ | Только с вашего Mac | Из интернета |
| Стоимость | Бесплатно | Обычно платно |
| Для чего | Обучение, разработка, тесты | Публичный сайт |
